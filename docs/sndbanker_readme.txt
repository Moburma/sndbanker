
Sound Banker - WAV to DAT sound bank creator (sndbanker)
  Created by Tomasz Lis; GNU General Public License
----------------------------------------
usage:
    sndbanker.exe [options] <filename>
where <filename> should be the input sounds list file, and [options] are:
    -v,--verbose             Verbose console output mode
    -o<file>,--output<file>  Output sound bank file name
